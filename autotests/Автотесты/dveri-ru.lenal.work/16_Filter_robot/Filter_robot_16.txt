1. Создаем папку под названием Filter_robot_16
2. Помещаем в папку Filter_robot_16 файл под названием filter.txt (берем из папки Final)
3. Помещаем в папку Filter_robot_16 файл под названием filter_robot.py (берем из папки Final)
4. Запускаем скрипт
5. В ходе выполнения скрипта будут отображены все теги robot
6. Если какой-то из url не содержится тег robot, то отобразится "wrong" и список url, в котором нет тега robot.
