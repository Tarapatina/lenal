1. Создаем папку под названием Canonical_13
2. Помещаем в папку Canonical_13 файл под названием list_new.txt (берем из папки Final)
3. Помещаем в папку Canonical_13 файл под названием canon.py (берем из папки Final)
4. Запускаем скрипт
5. В ходе выполнения скрипа проверяется наличие атрибута rel canonical в указанных url 
6. Если атрибут rel canonical присутствует - программа выдаст результат "ок"
7. Если атрибут rel canonical отсутствует - программа выдаст список url, в котором нет атрибута rel canonical
